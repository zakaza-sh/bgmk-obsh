# Система санитарного контроля общежития

Telegram Web App для управления санитарным контролем комнат в общежитии.

## Функционал

### Основные возможности
- **Этажи 2-9** с 15 блоками на каждом
- **Три типа комнат** в каждом блоке: Маленькая, Большая, Общее пространство
- **Система оценок 1-5** с цветовой индикацией (1-3 = красная/проблемная, 4-5 = зелёная)
- **Управление проживающими** с указанием старост блоков
- **Фильтрация и поиск** по датам, номерам блоков и проживающим
- **Роли пользователей**:
  - Администратор (полный доступ)
  - Староста этажа (доступ к своему этажу)
- **Расписание транспорта** с уведомлениями о срочности
- **Экспорт отчётов** в PDF формате
- **Отслеживание активности** старост для безопасности

## Технологии

### Backend
- FastAPI (Python)
- MongoDB
- JWT аутентификация

### Frontend

- Framer Motion (анимации)

- Sonner (toast уведомления)

## API Endpoints

### Аутентификация
- `POST /api/auth/login` - Вход в систему
- `POST /api/auth/register` - Регистрация (только для админа)
- `GET /api/auth/me` - Получить текущего пользователя

### Проживающие
- `POST /api/residents` - Добавить проживающего
- `GET /api/residents` - Список проживающих (с фильтрами)
- `PUT /api/residents/{id}` - Обновить проживающего
- `DELETE /api/residents/{id}` - Удалить проживающего

### Инспекции
- `POST /api/inspections` - Создать оценку
- `GET /api/inspections` - Список оценок (с фильтрами по дате, этажу, блоку)

### Блоки
- `GET /api/blocks/{floor}/{block}` - Информация о блоке

### Транспорт
- `GET /api/transport` - Расписание транспорта

### Админ
- `GET /api/admin/users` - Список пользователей
- `GET /api/admin/activity/{user_id}` - Активность пользователя
- `GET /api/admin/export/pdf` - Экспорт отчётов в PDF

# Обновления - Нумерация блоков и реальное расписание транспорта

## Изменения

### 1. Новая нумерация блоков ✅

Блоки теперь отображаются в формате **XXX** (этаж + номер блока):

**Примеры:**
- Этаж 2, блок 1 → **201**
- Этаж 2, блок 15 → **215**
- Этаж 3, блок 1 → **301**
- Этаж 3, блок 15 → **315**
- Этаж 9, блок 15 → **915**

**Где применяется:**
- Карточки блоков на странице этажа
- Заголовок страницы блока
- Список проживающих в админ-панели
- Поиск по номеру блока

### 2. Реальное расписание транспорта ✅

Интегрировано расписание для остановки **"Дом правосудия"** в Минске:

**Местоположение:**
- Адрес: ул. Семашко, 33
- Рядом с метро "Петровщина"
- ID остановки: 30505, 30506

**Маршруты автобусов:**

| Маршрут | Направление | Интервал |
|---------|-------------|----------|
| **103** | ДС Малиновка-4 — ДС Юго-Запад | ~12 мин |
| **57** | Семашко — ДС Восточная | ~15 мин |
| **38** | Автостанция Юго-Западная — Академика Карского, 44 | ~18 мин |
| **32С** | Экспресс маршрут | ~20 мин |

**Особенности:**
- ⏰ Динамическое расписание в зависимости от времени суток
- 🚦 Учет часа пик (7-9, 17-19): интервалы сокращаются на 30%
- 🌙 Ночное время (22-6): интервалы увеличиваются в 2 раза
- ⚠️ Отметка "срочно" если до прибытия < 5 минут
- 🎲 Реалистичная вариативность ±2 минуты

**server.py - транспорт:**

```python
    ("38", "Автостанция Юго-Западная"),
    ("32С", "Экспресс")
]

# Базовые интервалы
base_intervals = {
    "103": 12,
    "57": 15,
    "38": 18,
    "32С": 20
}

# Корректировка по времени суток
if 7 <= hour <= 9 or 17 <= hour <= 19:  # Час пик
    multiplier = 0.7
elif 22 <= hour or hour <= 6:  # Ночь
    multiplier = 2.0
else:  # Обычное время
    multiplier = 1.0
```

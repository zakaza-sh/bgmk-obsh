# Обновление: Выбор даты проверки

## Добавлено ✅

### Функция выбора даты при выставлении оценки

Теперь староста может выбрать дату проверки при выставлении оценки комнате.

**Интерфейс:**
1. Открыть комнату для оценки
2. Выбрать дату проверки (календарь)
3. Выбрать оценку 1-5
4. Сохранить

**Особенности:**
- По умолчанию установлена сегодняшняя дата
- Можно выбрать любую дату до сегодняшнего дня (не позднее)
- Удобный календарь для выбора
- Время проверки устанавливается автоматически (текущее время)

**Пример использования:**

1. **Проверка сегодня:**
   - Дата: сегодня (по умолчанию)
   - Оценка: 5
   - Сохранить

2. **Проверка была вчера:**
   - Изменить дату на вчера
   - Оценка: 4
   - Сохранить

3. **Проверка была неделю назад:**
   - Выбрать дату неделю назад
   - Оценка: 3
   - Сохранить

**Ограничения:**
- ❌ Нельзя выбрать будущую дату
- ✅ Можно выбрать любую прошлую дату
- ✅ По умолчанию - сегодня

**Формат даты:**
- В интерфейсе: стандартный календарь браузера
- В базе данных: ISO формат с временем (2026-03-10T14:11:02Z)

**API:**
```json
POST /api/inspections
{
  "floor": 2,
  "block": 5,
  "room_type": "small",
  "rating": 5,
  "inspection_date": "2026-03-10"  // YYYY-MM-DD
}
```

**Преимущества:**
- Староста может выставить оценку позже (если забыл)
- Корректная дата проверки в истории
- Точная статистика по датам
- Удобство для планирования

**Тестирование:**
✓ Оценка с сегодняшней датой работает
✓ Оценка со вчерашней датой работает
✓ Дата сохраняется корректно
✓ Ограничение на будущие даты работает
✓ Frontend и Backend синхронизированы

**Обновлённые файлы:**

- `/app/backend/server.py` - добавлен параметр inspection_date

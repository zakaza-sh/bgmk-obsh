# Обновления системы - Публичный доступ

## Что изменено

### Доступ для всех проживающих
Теперь приложение доступно **всем проживающим БЕЗ авторизации** для просмотра:
- ✅ Все этажи (2-9)
- ✅ Все блоки (1-15)
- ✅ Комнаты и их оценки
- ✅ Список проживающих
- ✅ Расписание транспорта

### Вход для старост
На главной странице добавлена **отдельная секция для входа старост**:
- 🔐 Кнопка "Вход для старост и администрации"
- 📝 Только авторизованные старосты могут выставлять оценки
- 👤 Каждый староста видит только свой этаж

### Архитектура доступа

#### Публичные страницы (без авторизации):
- `/` - Главная страница с выбором этажей
- `/floor/:floor` - Список блоков этажа
- `/floor/:floor/block/:block` - Детали блока с комнатами и оценками
- `/transport` - Расписание транспорта

#### Защищенные действия (требуют авторизации):
- Выставление оценок комнат
- Админ-панель (`/admin`)
- Управление проживающими
- Экспорт отчетов

## API Endpoints

### Публичные (без токена):
```bash
GET /api/blocks/{floor}/{block}     # Информация о блоке
GET /api/residents                   # Список проживающих
GET /api/inspections                 # История оценок
GET /api/transport                   # Расписание транспорта
GET /api/health                      # Статус системы
```

### Защищенные (требуют Bearer token):
```bash
POST /api/auth/login                 # Вход в систему
POST /api/auth/register              # Регистрация (только админ)
POST /api/residents                  # Добавить проживающего
PUT /api/residents/{id}              # Обновить проживающего
DELETE /api/residents/{id}           # Удалить проживающего
POST /api/inspections                # Создать оценку
GET /api/admin/*                     # Админ функции
```

## Тестирование без авторизации

```bash
# Получить информацию о блоке
curl https://sanitary-check.preview.emergentagent.com/api/blocks/2/1

# Получить список проживающих
curl https://sanitary-check.preview.emergentagent.com/api/residents?floor=2&block=1

# Получить оценки
curl https://sanitary-check.preview.emergentagent.com/api/inspections?floor=2

# Получить расписание транспорта
curl https://sanitary-check.preview.emergentagent.com/api/transport
```

## Преимущества новой архитектуры

1. **Прозрачность**: Все проживающие могут видеть состояние комнат
2. **Мотивация**: Публичные оценки стимулируют поддерживать чистоту
3. **Простота**: Не нужна регистрация для просмотра
4. **Безопасность**: Только старосты могут менять оценки
5. **Контроль**: Админ видит всю активность старост

## Сценарии использования

### Проживающий
1. Открывает приложение
2. Выбирает свой этаж
3. Находит свой блок
4. Видит оценку своей комнаты
5. Проверяет расписание транспорта

### Староста этажа
1. Открывает приложение
2. Нажимает "Вход для старост"
3. Вводит свой логин/пароль
4. Выбирает свой этаж
5. Проходит по блокам
6. Выставляет оценки комнатам
7. Система автоматически логирует его действия

### Администратор
1. Входит как админ
2. Видит все этажи
3. Открывает админ-панель
4. Управляет проживающими
5. Создаёт старост для этажей
6. Экспортирует отчёты в PDF
7. Отслеживает активность старост

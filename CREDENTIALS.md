## Система безопасности

### Отслеживание активности
- Все входы в систему логируются
- Сохраняется IP адрес и время входа
- История последних 50 входов для каждого пользователя
- Администратор может просмотреть активность любого старосты

### Защита от передачи паролей
- Если староста передаст свой пароль, это будет видно по:
  - Множественным входам в короткий период
  - Входам с разных IP адресов
  - Необычным временам активности

### Рекомендации по безопасности
1. **Регулярно проверяйте активность старост** через админ-панель
2. **Меняйте пароли** при подозрении на передачу
3. **Предупреждайте старост** о последствиях передачи доступа
4. **Используйте систему логирования** для выявления нарушений
